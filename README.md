# Bank-Analysis-Project-
This project focuses on analyzing bank data to gain useful insights and support decision-making. The analysis includes data cleaning, processing, and visualization to understand customer behavior, transactions, and trends. Tools like Excel/SQL/Power.bi/Tableau  were used to perform the analysis and create meaningful reports.

**In this project there are multiple files are mention include Cleaned And Uncleaned Data, Excel Dashboard , Power.bi Dashboard ,Sql Queries , Tableau Dahboard , PPT for this Dashboard , Bootcamp and also the  Demo Charts**
